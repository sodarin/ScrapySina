　　马上就要到2016年尾了，对于今年一整年的影像行业来说可谓颇为艰难与坎坷，除了依然要面对来自智能手机的强力冲击外，4月份的日本熊本地震也为整个影像市场带来不小的波动，所谓牵一发而动全身，尤其是不少依赖于索尼传感器的厂商来说，完全打乱了原本的产品计划。同时，由于下半年日元、欧元等国际货币汇率之间的变动，相机价格整体上涨。但好在也有不少惊喜和让大家喜闻乐见的产品出现，特别是微单市场又迎来一次新的洗牌。　　2016年恰逢奥运年，按照往年惯例，尼康与佳能在年初同时推出了顶级旗舰全画幅单反相机D5与EOS-1D X Mark II，在对焦与连拍上达到新的高度，为体育摄影带来全新的拍摄体验。于此同时，“超高速”相机也不仅仅是在单反相机中展开激烈的争夺，索尼、富士以及等厂商在2016年发布的旗舰微单也无一例外全都打出“速度”牌，甚至在消费级相机领域，走高速路线的相机也不在少数。下面，我们来总结下在过去的一年中各大相机厂商都做过些什么。　　对于佳能来说，2016可谓是举足轻重的一年。首先奥运年佳能毫无意外的推出了顶级旗舰单反相机EOS-1D X Mark II，虽然是意料之中，但EOS-1D X Mark II的出现仍然引起整个影像市场的关注。新一代EOS-1D系列的旗舰机型带来了更先进的对焦和更快的连拍的速度，当然4K佳能也并没有落下。EOS-1D X Mark II搭载第二代61点AF系统是一套具有革新意义的全新对焦系统，61个自动对焦点的覆盖面积中央区域纵向扩展约8.6%，周边区域纵向扩展约24%，并且中央AF对焦点的低亮度对焦敏感度提升至-3EV，更好的提升了弱光对焦性能。另外，无需预升反光板且在追踪自动对焦/自动曝光的情况下，EOS-1D X Mark II连拍速度最高可达约14张/秒，但更有趣的是，在实时取景下EOS-1D X Mark II能实现16张/秒的高速连拍，Lv取景比使用取景器取景获得的连拍张数更高，这的确很有意思。　　而实现实时取景16张/秒连拍的技术其实也不陌生，“全像素双核CMOS AF（Dual Pixel CMOS AF）”，这个最初应用于APS-C画幅中端机型EOS 70D上的对焦技术终于用到了全幅机身上，而且还是EOS-1D X Mark II这样的顶级机身。在实时显示拍摄时相机会进行相差检测自动对焦，并且对焦范围覆盖了80%的画面，比使用光学取景器的对焦范围更广。并且，在2016年佳能将“全像素双核CMOS AF”普及到全系列系产品，EOS 5D Mark IV便是其中之一。　　佳能EOS 5D系列算是佳能单反中最受摄影师和消费者关注的机型，这次EOS 5D Mark IV也确实捍卫了佳能“无敌”系列的好口碑，从画质到对焦、再到4K视频，基本上没有什么可挑剔的地方。全像素双核CMOS AF和触摸屏的加入让它变得不像是一台传统的“单反”，让去在实时取景下拍摄静态照片以及录制动态视频时都变得更加实用，EOS 5D Mark IV相比于前代产品来说虽算不上有什么质的飞跃，但个方面更加均衡，整体更加好用。同样受益于“全像素双核CMOS AF”的还有佳能在下班Photokina前期发布的EOS M5微单，更专业的操控和可靠的机身性能，也奏响了佳能准备在高端微单市场大展拳脚的前奏。　　尼康这几年的日子似乎都不太好过，从之前的掉渣门到今年新品一托再拖，相比起佳能的强势，不得不说尼康真的有点运气不佳，希望来年能奋起反击吧，毕竟整体实力还是摆在那的。产品方面，今年相机市场重磅新品的第一枪便是来自尼康D5，基本上尼康D5比起上代旗舰D4S从内到外都来了个大换血，153点AF对焦系统和最高ISO 3280000的感光度，这在单反诞生至今从未出现过，完全可以用变态来形容，同时在快门组件、液晶屏等硬件上全部有所突破。尼康D5的自动对焦系统采用了全新设计的 Multi-CAM 20K感应器模组，对焦点增加至153点（其中55点可选），而153点中99点为十字型对焦点，将单反的对焦极限又往前推进了一大步。D5的原生感光度最高为ISO 102400，扩展感光度由最低ISO 50到最高惊人的约ISO 3280000（Hi 5），是数码相机首次将感光度提升至百万级别。　　除了FX格式的旗舰D5，尼康在今年还有一款非常受欢迎的机型D500。这款DX格式的旗舰机型采用了与D5同规格的153点对焦系统，连拍速度达到了10张/秒，同样支持触摸屏和4K拍摄，可以说就是D5的缩小版，整体规格与性能应该是目前分全幅单反相机中之最，进一步提升了尼康在专业领域的地位。单反机型上，尼康还在下半年推出了两款常规升级的入门单反D3400和D5600，期待中的3位数全画幅机型（D8XX、D7XX）并没有迎来更新，这稍微让人感觉有点遗憾。　　至于DC产品，不得不提的自然是尼康原本打算力推的DL系列。从尼康的产品规划和结构来看，1系列微单似乎已经名存实亡，所以尼康在去年年初发布了同为1英寸CMOS的固定镜头相机DL产品线，虽然比起索尼、佳能来说在1英寸传感器DC市场起步晚，但其走的是搭配不同焦段镜头，走差异化路线，这当然也赢得了业界的一致好评，就等着尼康DL正式发售了。可谁知一方面由于相机软件的原因，另一方面熊本地震让传感器短缺无法满足相机生产，所以一拖再拖，直到年底也依然“只闻其声不见其影”。　　近几年一直专注刷机身的索尼在2016年终于开始在FE卡口的微单镜头上发力，前前后后发布了10余支全画幅微单镜头，大大扩充了全画幅微单镜头群的数量。其中比较瞩目的是索尼公布了全新的“GM”大师级镜头产品线，该系列镜头都采用最高标准的光学与工业设计，强调超高解析力和柔美焦外，其中FE 24-70mm f/2.8 GM和FE 70-200mm f/2.8 GM OSS的出现也预示着凑齐索尼全画幅微单“大三元”的日子不远了。　　而在机身方面，索尼倒是一反常态的没有太大动作，全画幅微单系列均无更新，传闻已久的顶级全幅A9也始终不见其踪影，不过这也意味着在2017年索尼或许会为大家带来更多惊喜。反倒是在APS-C画幅的E卡口微单上，索尼分别在年初和年尾发布了A6300与A6500，这两款相机发布时间并没有间隔太久，因为A6500并非A6300的替代升级，而是定位于APS-C旗舰，更像是之前NEX-7的继任者，基本上索尼相机的黑科技都用在了这款机器上，5轴防抖、4K视频等一个不落。比较令人意外的是，沉寂已久的索尼A卡口单电在今年总算又加入了新鲜血液，旗舰级单电A99II让不少A卡口用户吃了颗定心丸，说明索尼并没有放弃A卡口市场，并且传闻明年索尼A卡口镜头将迎来最大一次规模的更新。　　在手机冲击下的卡片机市场，索尼黑卡异军突起，从第一代RX100开始，索尼每一款黑卡相机都是销量和口碑双丰收，今年已经更新到RX100 V第五代，可以说，市面上没有一款比索尼RX100 V功能更全面的口袋卡片机了。但它也很贵，7000元的售价也注定了这并非一款符合大众市场的小DC，所以索尼很聪明，展开“机海战术”，第一代到第五代黑卡同步销售，不同价位都能找到合适的一款。　　以往谈起富士相机，更人更多的印象是复古、文艺、还有贵，但从另一个方面来看就是小众。但2016年是富士转变形象重要的一年，一方面主打“双旗舰”路线，年初新机X-Pro2正式登场，全新X-Trans CMOS III传感器与X Processor Pro将画质提升到了新的高度，对焦点从先前型号的49个扩展到77个，并且还可以使用对焦拨杆完成快速选择，众多的升级让期待已久的富士粉丝激动不已；六月份富士再发布主打高速对焦和连拍的专业机型X-T2，采用了胶片单反相机的外观，并且加入了4K视频录制功能，实现超凡图像质量、具备出众的操作性能以及传承复古经典相机的操控设计，同时也是富士真正意义上的第一款面向高端摄影师的速度型相机。双旗舰路线让富士继续稳固其高端的用户市场，并且由于出色的产品和富士转变了以往较保守的宣传方式，两款机型都获得非常不错的口碑和销量，大大增加了富士微单的普及度。　　另一方面，富士与索尼一样，在今年非常重视镜头的开发，尤其是针对偏入门和中端的用户，富士开启了“2.0”时代。以往富士微单镜头虽都具备超高的画质，尤其是定焦镜头也以大光圈标榜自己的特色，但在价格上并没有优势，所以富士为了扩展更多的用户群体，推出了F2.0光圈定焦镜头群，用更亲民的价格吸引更多人加入富士微单系统。　　当然，富士的野心不仅仅局限于APS-C画幅上，虽然消费者让富士出全画幅的呼声不算，但富士似乎并没有“领情”，反倒是非常出其不意的在photokina上发布了一款中画幅微单GFX 50S，相机采用全新的G卡口，搭载了一块5140万像素的中画幅CMOS传感器。不过，正式的上市时间和详细信息要在2017年一月份才会知晓。　　在越来越小众的M4/3系统市场，奥林巴斯在2016年让人留下印象的有两款产品，一款是复古到极致的PEN F，另一款则是专业到极致的OM-D E-M1 Mark II。奥林巴斯PEN-F搭载了一块M4/3画幅2030万像素的Live MOS传感器，并加入了236万像素的OLED电子取景器，与此同时，在PEN-F的机身背部我们还可以看到一块104万画点3英寸可翻转式LCD显示屏，而在OM-D E-M5 Mark II上取得成功的新一代五轴机身防抖技术，也被移植到了这款新机中。不过PEN-F最主要的特色还是自身的复古风格，相机机顶部分加入了大量造型精美的手动控制按钮和转盘，包括模式拨盘、曝光补偿拨盘以及创意模式拨盘。　　旗舰定位的E-M1 Mark II主要强调高速性、可靠性以及画质三方面。首先在高速性方面，E-M1 Mark II搭载了全新设计的自动对焦系统，其包含121个自动对焦点，且全部对焦点均为十字对焦点。此外，新机的最高连拍速度也来到了18张/秒，而如果你不需要使用自动对焦，相机的连拍速度则能够进一步提升至60张/秒全像素RAW文件连拍的水平。另外E-M1 Mark II还搭载了独有“专业抓拍”模式，在半按快门到全按快门之间的时间内，相机会预先拍摄14张照片，大大提高了捕捉动态瞬间的能力。　　松下是最早在消费类相机中加入4K录制的厂商，松下微单的4K拍摄能力也得到了业界的认可，但松下并没有只是在4K视频上做文章，而是通过4K功能扩展出不少独有的新功能。今年新发布的GX85和G80微单除了出色的4K视频拍摄能力，以及与4K相应的丰富模式，在此基础上又新增了全新的“4K裁剪”功能，在实用性上要比其他4K相机更出色。　　当然，2016年的相机并非只有以上这些，比如哈苏也在今年发布了针对消费群体的中画幅微单X1D，包括前面提到的富士GFX 50S，可以看到，2016年整个相机市场并没有继续只在“像素”上做太多文章，而是往更大画幅和更高画质上前进， 不过真正的较量也才刚刚开始。虽然今年的相机市场的发展有不少波折，但终归也为大家带来了不少优秀的产品，而展望2017年，我认为最值得期待的是尼康和索尼。尼康今年在全画幅领域基本上除了D5便默默无闻，一切的寂静都是为了明年的爆发，再加上明年是尼康100周年，有理由相信在这么重要的年份中尼康会有大动作；而索尼在今年牟足了劲推出新镜头，最大的可能是为了明年的重磅新品做铺垫，今年在全画幅机身上索尼同样停下了脚步，按照索尼更新机器的速度，明年也一定有好戏看。
