　　对于酷爱探险的驴友而言，一套专业实用的户外装备是不可或缺的，而关键时刻作为救命稻草的户外手持GPS更是必不可少的装备。近年来驴友在野外迷路甚至死亡的事件屡见不鲜，为避免此类悲剧发生驴友务必配备手持GPS。那么，什么样的手持GPS适合户外玩家使用呢！在这里就不得不提Garmin这个品牌了，作为全球最专业的GPS品牌，Garmin的手持类GPS一直是驴友户外探险最信赖的伙伴。近日，Garmin推出了最新款手持机GPS MAP 63sc，下面让我们看看被誉为新机皇的它在野外能为驴友提供哪些便利和帮助。　　作为一款户外用品，GPS MAP 63sc的品质是令人放心的，整个产品的做工非常扎实，它支持IPX7级防水，雨水环境下也可以正常使用。2.8英寸高清彩色显示屏，支持强光可视效果（强光下显示更清晰）。800万像素高清摄像头，可拍摄含有坐标数据的照片。在操作方式方面，将物理按键与触摸操作相结合，即便佩戴手套也可以轻松操作。　　GPS MAP 63sc内置GPS+GLONASS双星定位系统，机身上方设计了一个高灵敏度接收天线，可以在恶劣的野外环境里保持稳定的信号。它内置多轴电子罗盘，可显示驴友当前的方位，通过气压式高度计还能了解当前所处的海拔高度，以及气压和温度。　　作为一款户外手持机，GPS MAP 63sc最大的特点自然是导航了，它支持航点导航、航线导航、航迹返航三种方式。航点导航简单来说就是点对点导航，从A点出发抵达B点，至于应该选择如何行走，需要驴友利用经验和电子罗盘为辅助来进行导航。　　航线导航类似于常用的手机导航，通过GPS MAP 63sc搜索你想去的地方，GPS MAP 63sc就会为你规划一条线路，按照线路行进即可抵达目的地；而航迹返航功能简单来说就是原路返回，一旦在野外迷路也不必担心，GPS MAP 63sc会带你回到最初的位置。　　GPS MAP 63sc支持ANT+传输协议，无需使用电脑就能在野外将航点、航线和航迹无线传输给队友。此外，GPS MAP 63sc还提供了Live Track功能，可以将探险过程实时分享给亲朋好友，让他们在电脑上或手机端查看到你在地图上的位置，分享喜悦的同时让也亲朋好友放心。当然，驴友也可以在完成一次探险之旅之后将照片、视频和轨迹结合起来，制作完整的探险活动，让亲朋好友欣赏沿途的风景，一起领略这段了不起的探险之旅。　　在探险过程中像手机这种设备往往都会放在包里，这样就会容易错过重要的消息，为了方便驴友在探险过程中及时接收手机上的消息，GPS MAP 63sc可以通过蓝牙与手机连接，通过手持机就能第一时间获取手机上的来电、短信等重要消息，非常方便。
